# parcel-get-start

parcel 学习入门

## 开始

编译构建

```
# 执行 parcel watch src/index.html
> npm run watch
```

再新开一个命令窗口（终端)

```
# 启动 http-server
> npm start
```


## 开发

在浏览器中输入 [http://localhost:8083](http://localhost:8083)

## 展望

持续关注 `parcel` 官网文档更新，`parcel` 要变强大需要更多的插件，官方设想跟 postcss 一样吧，更多的让使用开发者自己定制。
